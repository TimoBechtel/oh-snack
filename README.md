## Install

### NPM:

```sh
npm install oh-snack
```

### CDN:

```html
<script src="https://unpkg.com/oh-snack/dist/index.umd.js"></script>
```

## Usage

### As module:

```javascript
import { snack } from 'oh-snack';
```

### Example:

```javascript
snack("Hi, I'm a snackbar notification 👋", 2500);
```
